# 1.0.0-rc.24

## 更新

* **工具箱**
  + 更新工具箱！内置很多有用的小工具！
  + 更新 **Pak解包/打包** 功能
* **下载确认** 可以显示游戏/修改器的 **新** 或 **推荐** 状态
* **效果** 为所有卡片的暗色模式添加边框

## 更改

* **主窗口** 更改导航视图标签页默认位置为 **顶部**
* **下载确认** 减小屏幕截图放大的大小 [#81](https://github.com/PvzLauncher/PvzLauncher/issues/81)

## 修复

* **管理** 修复了删除启动器后不会保存配置的问题
* **下载** 修复了下载完游戏/启动器后立即启动会导致崩溃的问题 [#82](https://github.com/PvzLauncher/PvzLauncher/issues/82)

## 移除

* **初始窗口** 移除初始窗口
